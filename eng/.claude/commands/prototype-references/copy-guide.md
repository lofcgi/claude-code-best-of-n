# Copy Writing Guide (Copywriting Formulas)

All text content in the prototype must follow this guide.
Remove the "AI-written smell" and generate professional copywriter-level content.

---

## Headline Formula

**Formula**: [Verb] + [Specific Target] + [Quantified Benefit]

| Bad (AI Slop) | Good (Pro Copy) | Why It's Good |
|---------------|-----------------|----------|
| "Innovative AI dubbing solution" | "33 languages, 4.2 seconds" | Specific numbers build trust |
| "Next-gen project management" | "Ship 2x faster" | Action + quantified benefit |
| "All-in-one marketing platform" | "From draft to revenue in 47 minutes" | Specific time promise |
| "Powerful data analytics" | "Find the signal in 2M rows" | Specific scale + action |

### Banned Words (AI Slop Markers)
- Revolutionary, Next-gen, All-in-one, Powerful, Cutting-edge
- Seamlessly, Effortlessly, Unlock, Leverage, Streamline
- "Revolutionize your ~"

### Recommended Patterns
- **Numeric Lead**: "98.5% uptime. Zero excuses."
- **Contrast**: "Less meetings. More shipping."
- **Direct**: "Turn recordings into text. Instantly."
- **Brand Verb**: Create a verb unique to the product (Notion: "wikify", Stripe: "accept")

## Feature Name Formula

**Formula**: [Brand Verb] + [Capability]

| Bad (Generic Noun) | Good (Branded Term) | Rule |
|----------------|------------------|------|
| "Voice synthesis" | "VoxSync Engine" | Proper noun + technical term |
| "Auto translation" | "LinguaFlow" | Compound brand name |
| "Real-time analytics" | "PulseMetrics" | Sensory + functional |
| "Collaboration tool" | "TeamForge" | Action + metaphor |
| "AI assistant" | "CoPilot Zero" | Brand + version feel |

### Feature Description Formula
- **1 sentence**: [What] + [How] + [Result]
- **15-20 word limit**
- **Include 1 specific number**

| Bad | Good |
|-----|------|
| "Powerful voice synthesis powered by AI" | "Generate natural voices in 33 languages in just 4.2 seconds" |
| "Efficient project management" | "Auto-rearrange tasks for a 142-person team with a single drag" |

## CTA Formula

**Formula**: [Action] + [Immediate Benefit]

| Bad (Default) | Good (Action+Benefit) | Rule |
|-----------|-----------------|------|
| "Get started" | "Try free for the first 10 minutes" | Time-limited benefit |
| "Sign up" | "Start in 30 seconds" | Speed promise |
| "Free trial" | "Start without a card" | Barrier removal |
| "Learn more" | "Watch the demo video" | Specific action |
| "Get Started" | "Start building free" | Action + free |
| "Sign Up" | "Create your first project" | Specific first step |
| "Try Now" | "See it in action — 2 min demo" | Time promise + action |

### CTA Placement Rules
- **Primary CTA**: glow box-shadow, large size (px-10 py-4)
- **Secondary CTA**: ghost/outline, placed next to Primary
- **Hero**: Must have 2 CTAs (Primary + Secondary)
- **Footer CTA**: Must have at least 1 CTA
- **Floating CTA**: Consider a bottom-fixed CTA based on scroll progress

## Numbers Formula

**Rule**: Specific > Rounded > None

| Bad (Rounded) | Good (Specific) | Rule |
|-----------|--------------|------|
| 10,000+ | 10,847 | Odd numbers feel more credible |
| 100% | 99.97% | Specific beats perfect |
| 99% | 98.5% | Decimals build trust |
| 1M | 1.2M | Add a decimal |
| 2x faster | 2.3x faster | Add a decimal |
| 50+ countries | 47 countries | Exact number |
| Thousands | 4,281 | Exact number |

### Trust Metrics Section Number Examples
```
10,847 → "Active teams"
98.5% → "Uptime SLA"
4.2s → "Average response"
47 → "Countries served"
```

### Where to Use Numbers
1. **Trust Metrics Section**: 4 count-up animations with NumberTicker
2. **Feature Descriptions**: 1 per feature
3. **Headlines**: Include 1 if possible
4. **Near CTAs**: 1 social proof number
5. **Footer**: In the format "Join X teams"

## Social Proof Formula

### Testimonial Structure
```
"{Specific result} — Thanks to {product name}, {specific time/cost savings}"
— {Name}, {Specific title} @ {Company name}
```

| Bad | Good |
|-----|------|
| "Great product" | "Onboarding time dropped from 3 weeks to 4 days" |
| "Very satisfied" | "Saving $12K per month — ROI achieved in 6 months" |
| "Highly recommend" | "A 142-person team voluntarily switched in the first week" |

### Logo Bar (Trust Bar)
- When real brand logos are unavailable: "Trusted by teams at companies like..."
- grayscale → hover:color effect
- Add motion with InfiniteMarquee

## Tone & Voice Matrix

| Project Type | Tone | Headline Style | Example |
|-------------|-----|---------------|------|
| B2B SaaS | Confident, concise | Numeric lead | "Ship 2x faster" |
| AI/ML Tools | Technical, precise | Capability lead | "Process 2M records in 4.2s" |
| Creative Tools | Inspiring, bold | Sensory lead | "Design at the speed of thought" |
| Productivity Apps | Practical, direct | Problem lead | "Never miss a deadline again" |
| Finance/Fintech | Trustworthy, stable | Safety lead | "99.97% uptime. Bank-grade security." |
| Education/EdTech | Friendly, supportive | Transformation lead | "From beginner to builder in 30 days" |

## Checklist (Final Copy Verification)

- □ 0 banned words in headlines
- □ All numbers are specific (0 rounded numbers)
- □ Feature names use brand-specific terms
- □ CTAs follow verb+benefit format
- □ Testimonials include specific results
- □ All descriptions are 20 words or fewer
- □ At least 1 feature includes a number
