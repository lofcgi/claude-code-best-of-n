# TaskFlow — 개인 작업 관리 앱

## 개요
개인 사용자가 일상적인 할 일을 카테고리와 우선순위로 만들고, 정리하고, 추적할 수 있는 심플한 웹 앱입니다.

## 대상 사용자
- 깔끔하고 집중할 수 있는 작업 관리 도구를 원하는 개인 사용자
- 팀 협업 기능 불필요 (싱글 유저 전용)

## 핵심 기능

### 1. 사용자 인증
- Google OAuth 로그인
- 브라우저 새로고침 시에도 세션 유지

### 2. 작업 CRUD
- 작업 생성: 제목, 설명(선택), 마감일(선택), 우선순위(높음/보통/낮음)
- 인라인 편집으로 작업 내용 수정
- 확인 후 작업 삭제
- 완료/미완료 토글

### 3. 작업 정리
- 카테고리: 사용자가 직접 카테고리 생성 가능 (예: "업무", "개인", "쇼핑")
- 카테고리, 우선순위, 완료 상태별 필터
- 마감일, 우선순위, 생성일 기준 정렬

### 4. 대시보드
- 오늘의 작업 뷰 (오늘 마감이거나 지연된 작업)
- 요약 통계: 전체 작업 수, 오늘 완료 수, 지연 작업 수
- 상단 빠른 작업 추가 입력

## 비기능 요구사항
- 모바일 반응형 디자인
- 페이지 로드 2초 이내
- 세션 간 데이터 유지 (서버 측 데이터베이스)

## 범위 밖
- 팀 협업 / 공유 작업
- 파일 첨부
- 캘린더 연동
- 푸시 알림
